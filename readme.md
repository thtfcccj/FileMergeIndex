### CCJ多功能文件合并器
  目标可执行文件下载地址：https://download.csdn.net/download/thtfcccj/81188414

#### 说明
  此应用可根据脚本文件不同，实现不同功能，主要有：
  * 资源文件合并功能:
     + 功能：将不同类型与大小的文件，合并生成为为一个bin文件，并在头部生成位置索引，供嵌入式开发调用。
     + 说明及示例：见“资源文件合并功能说明及示例”子目录
  * bin文件合并功能：
    + 将多个指定bin文件按给定位置合并为一个文件
     + 说明及示例：见bin文件合并功能说明及示例”子目录
