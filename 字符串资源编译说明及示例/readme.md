### 字符串资源编译说明及示例

#### 说明
  * 此脚本用于将不同长常量字符串以索此方式合成至一起
  * 主要用于如嵌入式开发中，将多个字符串合并为一个带索引头的bin文件。
  * 使用脚本方式加载文件，以用于后继增减文件方便，脚本文件格式见例“StringCompileCfgScript.txtt.txt”
  * 脚本文件中指定了起始存放位置，以用于后继使用UBIN或本软件进一步合并入应用。
  
#### 目标文件并按下述方式组织：
  目标文件以“数据头”+字符串数据流阵列“方式组织，
#### 数据头
  * 数据头长度：如脚本文件中指定最终文件为7个，则数据头长度为(7+1)*4(默认)=32个,最后多存一位，用于最后一个字符串数据计算长度。
  * 数据头格式：若配置为自动索引长度，则头部增加行长度，后跟索引位置(或为指针)，占用由定义决定
  * 字符串数据流阵列: 紧跟数据后，每个字符串数据之间没有间隔
  * 字符串数据流阵列大小：= 下个数据头值 - 当前数据头值
